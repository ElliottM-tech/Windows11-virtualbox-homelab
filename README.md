# Windows 11 VirtualBox Home Lab

## Project Overview

This repository documents the process of creating and configuring a Windows 11 virtual machine using Oracle VirtualBox.

The purpose of this project is to develop practical IT support skills by working with virtual machines, Windows installation, system configuration, updates, snapshots, and troubleshooting.

---

## Objectives

- Install Oracle VirtualBox
- Create a Windows 11 virtual machine
- Configure virtual hardware
- Install Windows 11
- Complete Windows setup
- Install Windows Updates
- Create snapshots
- Document troubleshooting
- Develop documentation skills

---

## Technologies Used

- Oracle VirtualBox
- Microsoft Windows 11
- Virtual Disk Image (VDI)

---

## Virtual Machine Configuration
